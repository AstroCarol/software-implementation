Task: Identificar Escopo do Projeto
O Analista de software, auxiliado pela pessoa responsável pelos aspectos contratuais do projeto (gerente de vendas) identifica claramente as principais funcionalidades que estão incluídas no escopo do projeto.

Dica: Identificar funcionalidades que estão fora do escopo também é muito valioso para esclarecer as diferenças de entendimento com seus clientes.
