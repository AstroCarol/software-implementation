Task: Verificar os Componenetes
Esta etapa consiste em uma revisão de código executada pelo programador, uma compilação e depuração. Verifique o programa em relação aos requisitos identificados para garantir que todas as funções necessárias do programa sejam implementadas. Siga a lista de verificação de revisão de código para encontrar todos os defeitos no programa. Ao fazer a revisão, marque os defeitos no código-fonte. Após a revisão do código, corrija todos os defeitos. Depois de concluir as correções, forneça uma listagem do programa de origem para o programa corrigido. Revise todas as correções para garantir que estejam corretas. Corrija quaisquer defeitos remanescentes. Deixe o computador verificar variáveis não declaradas, conflitos de nomenclatura e assim por diante, você pode ter feito isso antes da Revisão de Código também. Depois que a rotina for compilada, coloque-a no depurador e percorra cada linha de código. Certifique-se de que cada linha seja executada conforme o esperado. Você pode fazer isso após o teste da unidade com foco em encontrar a fonte de um defeito.
