Activity: Construção do Software
A atividade de Construção de Software desenvolve o código de software e os dados dos Componentes de Software identificados na Atividade anterior. A atividade fornece: Revisão do Plano do Projeto para determinar a atribuição da tarefa. Entendimento dos Componentes de Software identificados. Casos de Teste e Procedimentos de Teste para testes de unidade e integração. Componentes de software codificados e testes unitários aplicados.
