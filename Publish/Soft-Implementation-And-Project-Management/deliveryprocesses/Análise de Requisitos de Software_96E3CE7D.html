Activity: Análise de Requisitos de Software
A atividade de Análise de Requisitos de Software analisa os requisitos acordados do cliente e estabelece os requisitos de software do projeto validados. A atividade fornece: Revisão do Plano do Projeto para determinar a atribuição da tarefa. Seleção, análise e especificação dos requisitos do cliente. Acordo sobre os requisitos do cliente. Verificação e validação de requisitos.
