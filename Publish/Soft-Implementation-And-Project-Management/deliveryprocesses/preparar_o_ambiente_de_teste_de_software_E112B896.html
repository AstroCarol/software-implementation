Task: Preparar o Ambiente de Teste de Software
Durante esta etapa, o ambiente de teste, prontidão e disponibilidade de componentes (corrigidos ou baseline) a serem testados e dados que serão utilizados para esta execução são preparados. O objetivo é definir um ambiente controlado e independente para os testes. A preparação do ambiente de teste inclui, como exemplo: especificação de configuração de hardware, sistema operacional, navegador e configuração TCP/IP, especificação de software do sistema, mecanismo de base de dados e mecanismo de suporte de teste.
